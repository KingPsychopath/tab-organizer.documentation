Version 2.7

  • If you previously used bubble mode, it will revert back to popup mode. Just open the options page and change it again. You only need to do this once.
  • We now collect anonymous usage statistics. You can disable this in the options page.
  • You can now undo various actions. However, it is not possible to undo a "close tab" action. Please see issue 24.

  • New features:

    • Issue 5: The ability to undo actions.
    • Issue 25: Different color themes.
    • Issue 26: Show Tab Organizer in a tab, rather than a popup/bubble.

  • Bug fixes:

    • Issue 16: When a window's focus changes, sometimes it doesn't update.
    • Issue 22: Uncaught Error: chrome.windows is not supported in content scripts.
    • Issue 27: When Alt+Tabbing away from Tab Organizer, the window containers lose focus.

Version 2.6

  • Issue 5: Adding an undo bar. Doesn't do anything right now.
  • Issue 15: Abstracting chrome calls into Platform calls. This is the first step in porting Tab Organizer to other browsers.

  • Bug fixes:

    • Issue 18: tab organizer stopped functioning
    • Issue 19: Bubble mode shows an unnecessary horizontal scrollbar.

Version 2.5

  • The code that determines which window is focused should work a bit better now.
  • Adding a ton of unit tests for the Options API.

  • Bug fixes:

    • A selected tab that is in the queue looks the same as a non-selected tab that's in the queue.
    • When moving tabs around, sometimes the Shift key will get "stuck". Just click a second time to fix it.
    • When Alt+Tabbing away from Tab Organizer, the window containers lose focus.

Version 2.4

  • You can now use Alt+Click to close a tab. This is useful if your mouse doesn't have a middle button.

  • Keyboard:

    • The hotkey to open Tab Organizer now works on GMail.
    • You can now use the Up and Down arrow keys to select tabs while a container is focused.
    • You can now use (Ctrl/Shift/Alt)+(Space/Enter) to simulate (Ctrl/Shift/Alt)+Click.

  • Search box:

    • Adding the window:selected search query.

  • New features:

    • You can now open the popup in a bubble, if you prefer. It's off by default, but you can turn it on in the options page. Some things may be broken in bubble mode.

Version 2.3

  • We now show the tooltip when hovering over a tab's favicon.

  • Search box:

    • We now remember the last search query that you used.

  • Keyboard:

    • Shift key now only selects tabs that are visible.

  • New features:

    • The window that was last focused is now highlighted when Tab Organizer opens.

  • Bug fixes:

    • All windows share the same selection queue. Fixing this will require large structure changes.

Version 2.2

  • You can now use the Shift key to select multiple tabs at once.

  • Search box:

    • We now select the Search box when using Ctrl/⌘+F, rather than just focusing it.
    • Lots of cleanup and refactoring of the window: search query.

Version 2.1

  • Search box:

    • Adding the is:image search query.

  • Menu items:

    • Adding a "Move all selected" menu item.
    • Right-clicking within a window container now shows a context menu.

  • Options page:

    • Large changes to the options page. Things are now divided into groups based on what they change.
    • You can now use a keyboard shortcut (Ctrl+Shift+O) to open the Tab Organizer. This can be changed in the options page.

Version 2.0

  • We now show the tab's full title in a tooltip when hovering over the tab's text.

  • Search box:

    • The same:url search query now ignores a trailing slash at the end of the URL.

  • URL bar:

    • Brightening up the background some more, for readability.
    • Files (at the end of the URL) are now red. Directories are still blue.
    • The query and file parts of the URL are now bold.
    • file:// URLs are now colored. Before, there was no extra formatting.

Version 1.9

  • Changing the default popup size. Users with a custom size shouldn't be affected at all.
  • Adding the window-menu item "Rename window".
  • Reverting the change from Version 1.7: we now select a tab on click.
  • Several small visual tweaks.

Version 1.8

  • Changed the "close window" button into a drop-down menu ("close" is now an item in the menu).
  • Adding "New Tab", "Select all", and "Select none" to the drop-down menu.
  • Adding shading to the "close tab" button so it looks better.
  • Fixed it not scrolling to the selected tab.
  • Made the background color very slightly less red-hued.
  • Bunch of bug fixes for the search queries, especially window:.
  • Adding the is:selected search query.
  • Adding a ton more unit tests for the search box queries.

Version 1.7

  • We now use the GNU Lesser GPL rather than the X11 license.
  • Drag-n-drop now uses a placeholder, rather than moving the tabs right away. This also allows you to cancel the drag.
  • We now select a tab on mousedown rather than click. This helps a bit with accidental dragging.
  • Tried making the custom URL bar easier to read by brightening the background a bit.
  • Various small visual tweaks to make things look better.

Version 1.6

  • We still use native drag-n-drop, but it should be significantly better.
  • Renamed the is:duplicate search query to same:url.
  • same:url now ignores the hash fragment when finding duplicates.
  • Adding the same:title search query.
  • Adding the inurl: and intitle: search queries.
  • We now use our own custom URL bar in the lower-left corner.

Version 1.5

  • Left and Top options are handled differently. We now use 0% to mean "center", rather than 50%. Your settings should still be there, but the results will be different.
  • Fixed the "New Window" button not going to the New Tab page.
  • Vastly improved options page. Added on Reset and Cancel buttons.
  • Made the button(s) in the popup narrower, in case I add on more later.
  • Improved Options API. Just about everything is better, really.

Version 1.4

  • Massive changes to the Options API. May break past user's settings.
  • You can now adjust the position and size of the popup in the options page.

Version 1.3

  • We now update the popup's title to show how many tabs and windows you have.
  • Search now properly updates when a tab or window is changed.
  • Fixed the containers "flickering" when first opening the popup.

Version 1.2

  • Searching for a window (by label) is now case-insensitive.
  • The "close window" button appears on focus (like it used to). This reverts the change from Version 1.1.
  • Adding unit tests for the search code and String.prototype.escape.

Version 1.1

  • The "close window" button now appears on mouse hover, rather than on focus.
  • You can now change the title of the tab containers.
  • Added support for the Command key (⌘) on Mac for selecting multiple tabs.

Version 1.0.5

  • Added a close button to the right side of the hovered tab.
